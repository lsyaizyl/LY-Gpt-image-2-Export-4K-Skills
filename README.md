# LY-Gpt-image-2-Export-4K-Skills 简介

这是一个用于**通用图片高清导出与清晰度增强**的工作流 Skill。

它适用于海报、插画、产品图、人像照片、风景图、技术图、UI 截图、设计稿、AIGC 图片等多种类型图片。核心目标是在**不改变原图宽高比例、不裁切、不拉伸、不破坏构图**的前提下，对图片进行等比高清放大、细节增强、锐化处理和最终导出。

本 Skill 默认不强制套用固定尺寸，例如 2160×3840 或 3840×2160，而是根据原图实际比例进行高清化处理。用户没有明确指定尺寸时，可按 2 倍、3 倍或目标长边进行等比放大；用户明确指定尺寸时，再切换为固定尺寸导出流程。

同时，本 Skill 也包含针对不同图片类型的增强重点，例如文字海报优先保证文字清晰，产品图优先保证轮廓和材质，人像图优先保证自然五官和皮肤质感，技术图优先保证线条、标注和结构准确。

适合用于构建一个稳定、可复用的图片高清导出流程，避免出现图片被错误拉伸、裁切、比例改变、文字乱码未修复、导出尺寸不说明等问题。
