# Skill：通用图片高清增强、文字修复与等比导出工作流 v2

## Skill 简介

这是一个用于任意图片的高清增强、文字修复、细节优化与等比导出的通用工作流 Skill。

适用于：

- 海报
- 插画
- 产品图
- 人像照片
- 风景图
- 电商主图
- UI 截图
- 信息图
- 技术图
- 设计稿
- AIGC 生成图
- 含文字图片

核心目标：

> 在不改变原图宽高比例、不裁切、不拉伸、不破坏构图的前提下，减少模糊、修复文字乱码、增强细节，并导出更高清的图片文件。

---

## 1. 核心原则

### 1.1 默认保持原图比例

除非用户明确指定固定尺寸，否则默认保持原图宽高比例。

禁止默认执行：

- 拉伸
- 压缩
- 裁切
- 横图改竖图
- 竖图改横图
- 强制改成 9:16
- 强制改成 16:9
- 强制改成 1:1

默认流程：

```text
保持比例 → 修复问题 → 提高清晰度 → 等比放大 → 导出文件
```

---

### 1.2 先修复，再放大

如果图片存在模糊、乱码、压缩痕迹、边缘破损等问题，不要直接放大。

正确顺序：

```text
识别问题 → 修复画面 → 修复文字 → 生成清晰底图 → 等比放大 → 锐化增强 → 验证尺寸 → 输出文件
```

---

### 1.3 文字是最高风险区域

AI 生成图片时，小字、中文、表格、参数、标签、密集说明最容易出错。

处理原则：

1. 标题必须正确。
2. 关键文字必须正确。
3. 数字、单位、参数必须正确。
4. 小字如果无法保证准确，宁愿简化或留白。
5. 不要生成伪中文、乱码、随机符号。
6. 大量文字图片应优先使用“底图修复 + 真实文本叠加”的方案。

## 2. 判断用户需求

### 2.1 等比高清导出

当用户说：

- “提高清晰度”
- “高清一点”
- “放大一点”
- “保持比例”
- “不要变形”
- “不要裁切”
- “按原图比例”
- “输出高清版”

默认处理方式：

```text
保持原图宽高比例，按倍数或目标长边等比放大。
```

---

### 2.2 固定尺寸导出

只有用户明确指定固定尺寸时，才使用固定尺寸导出，例如：

- “导出 2160×3840”
- “改成 1920×1080”
- “我要 3000×3000”
- “做成 16:9 横版”
- “裁成 1:1 方图”

固定尺寸可能涉及裁切、留白或重构图，需要明确说明。

---

### 2.3 文字修复后导出

当用户提到：

- 乱码
- 错字
- 伪中文
- 小字糊
- 参数看不清
- 标题不对

应先修复文字，再导出高清版本。

## 3. 通用高清增强提示词

```text
请基于参考图制作一版更高清、更清晰、更干净的版本。

总体要求：
1. 保持原图主体、构图、宽高比例、画面方向、色彩关系和整体风格不变。
2. 不裁切、不拉伸、不改变画面比例。
3. 提高清晰度、边缘锐度、细节密度和整体质感。
4. 主体边缘要干净自然，避免毛边、锯齿、残影和模糊边。
5. 背景纹理、阴影、高光和材质细节要更清楚。
6. 不要随意改变主体身份、物体结构、服装、颜色、品牌标识或重要细节。
7. 不要增加无关元素。
8. 输出效果应像高清成品图，而不是简单模糊放大图。
```

---

## 4. 防模糊增强提示词

当用户反馈“还是模糊”“细节糊”“不够清晰”时，使用这个版本。

```text
请对参考图进行高清重建和清晰度增强，而不是简单放大。

重点优化：
1. 提升主体边缘锐度，避免软焦和雾化感。
2. 增强中小细节的可辨识度，不要把细节糊成纹理块。
3. 清理压缩痕迹、噪点、虚边、残影和锯齿。
4. 强化线条、边框、轮廓、材质纹理、阴影层次和高光细节。
5. 保持原图比例、构图、主体位置和视觉风格不变。
6. 不要重构成另一张图，不要改变关键物体和内容。
7. 输出应接近高清印刷图或高质量数字成品图。
```

## 5. 文字修复提示词

### 5.1 少量文字修复

适用于标题、Logo、少量标签。

```text
请修复图片中的文字，使其清晰、准确、可读，同时保持原图构图、风格、颜色、主体和宽高比例不变。

文字要求：
1. 中文必须是标准简体中文。
2. 不允许出现乱码、伪中文、错字、缺笔画或不可读字符。
3. 英文、数字、符号和单位必须清晰准确。
4. 保持原有字体气质和排版位置。
5. 不要编造原图没有的信息。

需要保留或替换的文字：
【粘贴文字清单】
```

---

### 5.2 大量文字图片处理

如果图片包含大量小字、表格、参数、菜单、标签、说明，不建议让图像生成模型直接修复所有文字。

推荐流程：

```text
1. 生成干净清晰的视觉底图。
2. 弱化或清除原有乱码区域。
3. 用真实文本重新叠加关键文字。
4. 再做等比高清导出。
```

提示词：

```text
请基于参考图生成一版干净清晰的视觉底图，保持原图构图、比例、主体和风格不变。

重要要求：
1. 对大标题和关键文字尽量修复为准确文本。
2. 对过小、密集、难以正确生成的文字区域，不要生成乱码。
3. 可以将无法保证准确的小字区域处理为干净的空白文字框、浅色占位线或简化标注。
4. 保持排版区域和信息层级，为后续叠加真实文字保留空间。
5. 不要用伪中文填充小字区域。
6. 不要生成不可读的假文字。
```

---

### 5.3 中文防乱码约束

```text
中文文字必须满足以下要求：
1. 使用标准简体中文字符。
2. 不使用伪中文、异体乱码、残缺笔画或随机符号。
3. 每一个标题、标签、参数和说明必须可读。
4. 如果无法保证小字准确，请减少小字数量或留出干净文本区域，不要生成假字。
5. 数字、单位、标点必须准确，例如：mm、kg、kW、%、℃、m²。
6. 不要混入无意义字符。
```

## 7. 等比高清导出策略

### 7.1 默认放大规则

如果用户没有指定尺寸：

- 普通高清：2 倍等比放大
- 更高清：3 倍等比放大
- 接近 4K：最长边提升到约 3840 像素
- 海报级：最长边提升到 4000–6000 像素

不要盲目承诺“原生 4K”。

---

### 7.2 按目标长边导出

适合：

- “接近 4K”
- “长边 4000”
- “适合打印”
- “大图导出”

逻辑：

```text
保持原图比例不变，将最长边提升到目标像素，另一边自动计算。
```

## 8. Python 等比导出模板

### 8.1 按倍数等比放大

```python
from PIL import Image, ImageFilter

src = "/mnt/data/input.png"
out = "/mnt/data/output_upscaled.png"

scale = 2.0

img = Image.open(src).convert("RGBA")
w, h = img.size

target_size = (round(w * scale), round(h * scale))

resized = img.resize(target_size, Image.Resampling.LANCZOS)

resized = resized.filter(
    ImageFilter.UnsharpMask(
        radius=1.4,
        percent=140,
        threshold=2
    )
)

resized.save(out, format="PNG", optimize=True)

print("原图尺寸：", (w, h))
print("导出尺寸：", target_size)
print("输出文件：", out)
```

---

### 8.2 按目标长边等比放大

```python
from PIL import Image, ImageFilter

src = "/mnt/data/input.png"
out = "/mnt/data/output_long_edge.png"

target_long_edge = 3840

img = Image.open(src).convert("RGBA")
w, h = img.size

scale = target_long_edge / max(w, h)
target_size = (round(w * scale), round(h * scale))

resized = img.resize(target_size, Image.Resampling.LANCZOS)

resized = resized.filter(
    ImageFilter.UnsharpMask(
        radius=1.4,
        percent=140,
        threshold=2
    )
)

resized.save(out, format="PNG", optimize=True)

print("原图尺寸：", (w, h))
print("导出尺寸：", target_size)
print("输出文件：", out)
```

---

### 8.3 质量增强版导出

适合模糊图、文字图、线稿图。

```python
from PIL import Image, ImageFilter, ImageEnhance

src = "/mnt/data/input.png"
out = "/mnt/data/output_enhanced.png"

target_long_edge = 3840

img = Image.open(src).convert("RGBA")
w, h = img.size

scale = target_long_edge / max(w, h)
target_size = (round(w * scale), round(h * scale))

resized = img.resize(target_size, Image.Resampling.LANCZOS)

resized = resized.filter(
    ImageFilter.UnsharpMask(
        radius=1.2,
        percent=150,
        threshold=2
    )
)

resized = ImageEnhance.Contrast(resized).enhance(1.05)
resized = ImageEnhance.Sharpness(resized).enhance(1.08)

resized.save(out, format="PNG", optimize=True)

print("原图尺寸：", (w, h))
print("导出尺寸：", target_size)
print("输出文件：", out)
```

## 9. 大量文字图片的推荐方案

如果用户要求文字准确，且图片文字很多，优先使用：

```text
视觉底图修复 + 真实文本叠加 + 等比高清导出
```

不要只依赖 AI 图片模型直接生成所有中文小字。

可用工具：

- PIL
- SVG
- HTML/CSS
- PPT
- Figma
- Canva
- 设计工具文本图层

优点：

- 中文真实
- 参数准确
- 无乱码
- 可编辑性更好
- 后续修改更方便

## 12. 失败处理策略

### 12.1 如果仍然模糊

不要重复同样提示词。

改用：

```text
高清重建 + 细节增强 + 锐化后处理
```

并降低一次性生成复杂度。

---

### 12.2 如果仍然乱码

不要继续让图像模型反复修全部小字。

改用：

```text
底图修复 + 真实文字叠加
```

---

### 12.3 如果画面被改坏

回到上一版较好的图，降低改动幅度。

提示词改为：

```text
只增强清晰度，不重构画面，不改变主体和版式。
```
